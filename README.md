# LEXUS HUB

Lexus Hub is a camlock script made to help you lock onto players easily. It works for both mobile and PC and is made to be simple, smooth, and easy to use.

# OVERVIEW
Lexus Hub is built to run well on any device. Everything is made to be stable, easy to control, and simple to understand. The goal is to give users a clean camlock with the features they need without extra stuff.

# FEATURES

## Camlock
A smooth camlock that follows players and works with custom prediction.

## Strafe System
Adds better movement so you can strafe more smoothly.

## ESP System
Shows player outlines and info so you can see people clearly.

## Mobile Compatibility
Works on mobile with easy touches and a clean layout.

## PC Compatibility
Works on keyboard and mouse with simple controls.

## Credits Section
Shows the people who helped or supported the project.

## Prediction Engine
Lets you change prediction so the camlock can fit your aim or playstyle.
